------------------------
 Steps to run SASS
------------------------
1.Create package.json typing in CMD:
 
$ npm init

2.Install the dependent packages, in CMD type: 

$ npm install

3.Install gulp globally, open Command prompt and type: 

$ npm install -g gulp

4.Install gulp locally type: 

$ npm install --save-dev gulp

5.Check Gulp version type(optional):

$ gulp -v

6.To use Gulp for compiling SCSS file we need to install a plugin:

$ npm install --save-dev gulp-sass

NOTE: Above 6 steps need to run only once for each project

------------------------
Compile SASS files
------------------------

7.Run following command to compile SASS and JS file

$ gulp
